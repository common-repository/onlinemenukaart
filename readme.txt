== Description ==

Onlinemenukaart is a SaaS that allows you to create a menu card for your business. With this plugin, you'll be able to  display the menucard on your Wordpress website.

= Features =

* Automatically connect your Onlinemenukaart account with your Wordpress site
* Gutenberg Block
* Shortcodes

More information about Onlinemenukaart can be found on our [website](https://onlinemenukaart.nl).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `onlinemenukaart` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Onlinemenukaart settings page and follow the instructions.
